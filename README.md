## :book: 학습 내용

### Gatsby란?

- 리액트에 새로운 규칙을 추가하는 리액트 기반의 프레임워크.(라이브러리X)

### 프로젝트 생성

#### 1. gatsby 전역 설치

> ❗node.js가 최신으로 설치되어 있어야 한다.

```
npm i -g gatsby-cli
```

#### 2. 프로젝트를 생성할 폴더로 이동 후, gatsby 실행

> ❗해당 프로젝트를 포함한 상위 폴더명은 전부 영문이어야 한다.

```
gatsby new
```

#### 3. 프로젝트 폴더로 이동 후, 프로젝트 실행

```
cd <프로젝트 폴더명>
npm run develop
```
